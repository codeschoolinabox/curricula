# Teaching Tips: Chapter 0

## Design Notes

- **IDE requirement**: Should the curriculum require a specific IDE (VSCode) from the start, or allow learners to use any text editor? VSCode provides useful features (formatting, linting, extensions) but adds setup complexity for absolute beginners.
- **Study Lenses dependency**: The curriculum assumes Study Lenses for exercises. Should Chapter 0 include setup instructions for the Study Lenses environment, or is that handled separately?
- **This chapter is the most open-ended** in the outline. The exact content and exercises need to be fleshed out as the rest of the curriculum solidifies — the orientation should reflect what's actually in the course.
