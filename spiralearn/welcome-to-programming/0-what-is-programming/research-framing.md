## Rhetorical Model Origins

The "rhetorical triangle" framing — source code as a text addressing three
audiences (developers, computer, users) — adapts classical rhetoric
(writer/audience/text) to programming education. This structural backbone drives
the entire curriculum's chapter progression.

_Citations and evidence to be added as research committee develops the evidence
base for Chapter 0's design choices._

## Comprehension-First Pedagogy

The comprehension-before-production principle — understanding code before
writing it — aligns with established reading-before-writing pedagogies across
disciplines.
