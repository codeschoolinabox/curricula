# The Rhetorical Model: Code Audiences

Programming is introduced through a rhetorical lens. Source code is a text that
simultaneously communicates with three audiences:

1. **Developers** — other humans who read, review, and modify the code. They see
   comments, variable names, formatting, and structure.
2. **A Computer** — the machine that executes instructions. It processes data
   through a cycle of state → instruction → state → instruction → ...
3. **Users** — people who interact with the running program through its
   interface, without seeing the source code.

This "rhetorical triangle" adapted from classical rhetoric
(writer/audience/text) provides the structural backbone of the entire
curriculum. Each subsequent chapter adds one of these audiences to the learner's
awareness.
