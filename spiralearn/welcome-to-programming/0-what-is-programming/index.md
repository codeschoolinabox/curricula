# Chapter 0: What is Programming?

> Before writing a single line of code, learners orient themselves to what
> programming is.

## Overview

This chapter introduces programming as an act of collaborative communication —
not as a purely technical skill, but as a form of writing that addresses three
distinct audiences through source code. This framing shapes everything that
follows.

No language features are introduced here. The goal is purely conceptual
orientation. For practical setup and course mechanics, see
[Getting Started](../-1-getting-started/).
