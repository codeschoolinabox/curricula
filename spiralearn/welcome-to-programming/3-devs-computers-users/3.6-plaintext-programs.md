# 3.6: Plaintext Programs

A fun and challenging interlude. Learners study and write programs in a text editor with **no syntax highlighting, no hints, no autocomplete**. The purpose is to remind ourselves that code is just text — and to practice relying purely on our own understanding.

## Language Features

- (nothing new)

## Skills and Objectives

- 🐣 Reading and understanding programs without IDE assistance
- 🐣 Writing syntactically correct code without autocomplete or error highlighting
- 🐥 Appreciating what IDE tools do for us (by experiencing their absence)
