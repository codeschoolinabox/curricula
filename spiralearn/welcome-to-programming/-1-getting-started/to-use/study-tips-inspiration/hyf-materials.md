# HYF Be Materials

Each module requires creating a check-in issue to track progress through
Suggested Study resources, Sunday Prep materials, and individual Exercises, with
Thursday Check-Ins documented in the issue.

## Module Exercises

Emoji difficulty levels guide study time allocation:

- 🥚: Required foundational material; comfortable completion expected with time
- 🐣: Started exercises; completable with effort despite difficulty
- 🐥: Studied examples; big-picture understanding achieved but exercise
  confidence may be limited
- 🐔: Related but non-essential; pursue after mastering other levels

Exercises focus on specific skills through smaller, repetitive problems with
tests.

Three exercise types exist:

1. **Theory**: Deep technology exploration with execution visualization;
   essential for debugging and testing
2. **Practice**: Bite-sized coding challenges applying learned technologies to
   small problems
3. **Integration**: Tiny projects integrating multiple skills into realistic
   scenarios

## Suggested Study

Optional resource lists support each module's learning objectives. Students may
substitute preferred resources or contribute new recommendations via pull
request.

## Sunday Prep

Organized topic lists precede each Sunday lesson, requiring familiarity before
class.

## Extra Support

Students needing support should contact via Discord or Thursday Check-in.
