# Learning Objective Priorities

Introduces the emoji priority system used throughout:

- 🥚 **Required** — base skills for this module and the next
- 🐣 **Started** — completable with more time and effort
- 🐥 **Studied** — big-picture understanding, may not be confident with
  exercises
- 🐔 **Bonus** — extension material
