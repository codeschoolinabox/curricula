## Empirical-First Complexity Teaching

The "complexity is counting" methodology — learners count steps before learning
notation — follows an empirical-before-formal progression. Big O emerges from
observed patterns, not from definitions.

## Space-Time Asymmetry

In 2025,
[Ryan Williams proved](https://blog.computationalcomplexity.org/2025/02/you-need-much-less-memory-than-time.html)
that any algorithm that runs in time t(n) can be simulated using only ~√t(n)
space. Space is provably, fundamentally cheaper than time. The intuition: you
can reuse memory, but you can't reuse time.
